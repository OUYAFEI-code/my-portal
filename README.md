# 个人门户大厅

## 使用说明

### 头像设置
将你的头像图片命名为 `avatar.png` 并放在此目录下,网页会自动加载。

支持的图片格式:
- PNG (推荐)
- JPG/JPEG
- WebP

建议尺寸: 400x400 像素或更大的正方形图片

### 项目链接
- **个人简历**: https://resume.happy-ou-ai.top/
- **复仇者联盟机场大战挑战**: https://game.happy-ou-ai.top/

## 文件结构
```
my-portal/
├── index.html          # 主页面
├── avatar.png          # 个人头像 (需要手动添加)
└── README.md           # 说明文档
```

## 本地预览
直接在浏览器中打开 `index.html` 文件即可预览。
